> Design limpo, responsivo e acessível.

## 📸 Demonstração

🔗 [Acesse o site ao vivo](https://akinonnim.github.io/selfsite)

## 🚀 Funcionalidades

- ✅ Navegação fixa (sticky) com links suaves para as seções
- ✅ Seção "Sobre" com apresentação pessoal e foto de perfil
- ✅ Experiência profissional em cards interativos
- ✅ Grade de tecnologias (frontend, backend, infraestrutura)
- ✅ Projetos relevantes com links para demo e GitHub
- ✅ Contatos (LinkedIn, GitHub, E-mail) em cards com hover
- ✅ Design responsivo (desktop, tablet e mobile)
- ✅ Animações e transições sutis (hover, translate, box-shadow)
- ✅ Paleta de cores terrosa e gradiente de fundo

## 🛠️ Tecnologias Utilizadas

- **HTML5** – Estrutura semântica e acessível
- **CSS3** – Flexbox, Grid, variáveis CSS, transições, gradientes, media queries
- **Google Fonts** – Ubuntu Mono e Segoe UI (fallback)
